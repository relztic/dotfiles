Custom terminal setup to streamline my development workflow.  
Synchronized using symbolic links.

## Installation

```sh
# config
ln -s ~/github.com/relztic/dotfiles/.config ~/.config

# commands
ln -s ~/github.com/relztic/dotfiles/.bashrc ~/.bashrc
ln -s ~/github.com/relztic/dotfiles/.zshrc ~/.zshrc
```
